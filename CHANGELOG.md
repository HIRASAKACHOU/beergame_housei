# ビールゲーム - チェンジログ

## [0.8] - 2025-11-16

### 追加機能
- **AI パラメータ化システム**: AIの係数が硬コードされなくなり、動的に設定可能に
- `defaultParams` オブジェクトによるAIパラメータの集中管理
- `aiParams` オブジェクトをゲーム初期化時に受け渡し可能

### 改善
- AI戦略の柔軟性向上：panic型, safe型, calm型のパラメータを独立管理
- UI命名の統一：積極型 → 冷静型（calm型）に改名
- 将来の需要曲線変動への対応設計

### 変更
- `AIStrategy.panic()`, `safe()` メソッドが `params` オブジェクトを受け取る
- `AIStrategy.calm()` メソッド追加（`aggressive` は向後互換のため保持）
- `makeDecision()` が `aiParams` パラメータをサポート
- `BeerGame.initialize()` が `aiParams` 設定を受け入れる

### 技術詳細
- **panic型**: demandMultiplier: 1.5, randomFactor: 0.1
- **safe型**: safetyStock: 8, demandMultiplier: 1.0
- **calm型**: demandMultiplier: 0.9, randomFactor: 0.05

### ファイル変更
- `index.html`: 全4つのAI役割選択を冷静型に対応
- `game.js`: AIStrategy クラスを完全に再構築、パラメータ化対応
- `AI_PARAMETERIZATION.md`: 新規追加（実装詳細ドキュメント）

## バージョニングポリシー

- **メジャーバージョン (1.0以降)**: 大幅な機能追加や設計変更時（管理者のみ更新）
- **マイナーバージョン (0.x)**: 通常の機能追加・改善・バグ修正
- **パッチバージョン**: 現在は使用していません

## 更新時のガイドライン

1. **日本語で記述**: すべての更新を日本語で記載
2. **バージョン番号**: `v0.x` 形式を維持（大版本は0のまま）
3. **コミットメッセージ**: `v0.x: 変更内容` 形式
4. **詳細**: 以下をカバー
   - 追加機能（Additions）
   - 改善項目（Improvements）
   - 変更点（Changes）
   - ファイル修正リスト（File Changes）

5. **Git操作**: コミット後、自動的にこのファイルを更新
