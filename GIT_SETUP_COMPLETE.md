## 🎯 今後のワークフロー

### コミット前チェックリスト
1. ✅ 変更内容を確認: `git diff`
2. ✅ ステージング: `git add -A`
3. ✅ 日本語でコミット: `git commit -m "v0.X: ..."`
4. ✅ プッシュ: `git push origin main`
5. ✅ CHANGELOG.md を更新

### コミットメッセージ形式（日本語）
```
v0.X: 主要な変更内容

**追加機能:**
- 項目1
- 項目2

**改善:**
- 項目1

**ファイル修正:**
- game.js: 内容
- index.html: 内容
```
