## 🎮 ゲームのポイント

### 出荷決定時
- 画面左の「在庫エリア」の数字を参考
- 「最大」ボタンで最大出荷量に設定可能
- 「出荷確認」で決定

### 発注入力時
- 画面右の「発注エリア」で発注量を入力
- 在庫と需要のバランスを考慮
- 「発注確認」で決定

### 運送アニメーション
- 入荷処理の商品は**黄色**で上下浮動
- 運送中の商品は**青色**で左右移動
- 次週到着予定は**緑色**でパルス

## 💡 新機能の特徴

### 1. トースト通知
```
┌─────────────────┐
│ 第X週が開始    │
│ しました       │
└─────────────────┘
    ↓ 2秒待機
   消滅
```

### 2. 入荷アニメーション
```
商品
  ↗ ↖    ← 上下浮動
 ↗   ↖   ← スケール変化
↗     ↖  ← 黄色背景
```

### 3. 運送アニメーション
```
商品 →→→ 
  ↘ ↙    ← 左右移動
   ↘ ↙   ← スケール変化
  ↘   ↙  ← 青色背景
```

## 🐛 トラブルシューティング

### 症状: アニメーションが見えない
```
対処:
1. ブラウザキャッシュをクリア (Ctrl+Shift+Delete)
2. ページをリロード (Ctrl+R または F5)
3. 別のブラウザで試す
```

### 症状: 日本語が文字化けしている
```
対処:
1. ブラウザの文字コード設定を確認
2. ページエンコーディングをUTF-8に設定
3. 別のブラウザで試す
```

### 症状: ゲームが固まる
```
対処:
1. ブラウザを再起動
2. キャッシュをクリア
3. コンソールでエラー確認 (F12)
```
